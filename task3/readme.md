        Необходимо создавать только одну запись в базе данных для каждого уникального веб-хука.

    # Использовать external_id из веб-хука как уникальный идентификатор.

        в базе данных через _getDealByExternalId.
    # Если запись существует, не сохраняем сделку (игнорируем дубликаты).

        дубликатов при высокой нагрузке.
